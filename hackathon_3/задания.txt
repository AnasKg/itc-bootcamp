1. Получить всех студентов по API http://206.189.44.36:8900/students/
    и вывести на экран имя, возраст, номер телефона и вывести сообщение "Совершеннолетний"
    если возраст старше 18, иначе "Несовершеннолетний"

    Для справки, структура студента, пример:
      {
        "id": 1,
        "name": "Bob",
        "age": 20,
        "group": "Python-3",
        "phone": "+996707286538",
        "email": "test@gmail.com",
        "date_joined": "2021-05-28T17:45:42.211688+06:00",
        "updated": "2021-05-28T17:45:42.211712+06:00",
        "created_by": "Anas"
    }

2. Добавить нового студента по API http://206.189.44.36:8900/students/
    и создателем(created_by) указать "Команда {№ вашей команды}"
    получить id нового студента(подсказка: Запрос добавления студента возвращает результат)
    и сделать запрос для получения одного студента по id которую вы получили
    и вывести id, имя, возраст и группу студента

3. Удалить любого студента и если статус код (я этого вам не рассказывал, но гугл вам в помощь) отклика(response)
    равен 204 то вывести сообщение "Успешно удалено", иначе вывести текст отклика


4. Перейти по ссылке на сайт https://softech.kg, найти раздел "Рекомендуемые",
    изучить его html код, задача спарсить название товара, цену и короткое описание. 
    Записать полученные товары в массив(list), где каждый элемент является dictionary
    Пример:
        [
        {'name': 'Стоп фары на крыло Mijia M365', 'description': '&n.....', 'price': '500 сом'},
        ]
    Спавка: strip() функция string, которая удаляет лишние пробелы, отступы


5. Спарсить айфоны по ссылке https://softech.kg/phones/apple-smartphone/, получить следующие данные о телефоне:
    * название
    * Цена
    * Производитель
    * Модель
    * Артикул

    Все эти данные можно получить в детальной информации товара(отдельная страница),
    Спарсенные телфоны записать в массив
